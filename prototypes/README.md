# Prototypes Directory

このディレクトリは、新機能の概念実装とプロトタイプ開発のためのスペースです。

## 目的

- 新しいアーキテクチャの概念検証
- 実験的な実装のテスト
- 本番実装前の設計検証
- チーム内での技術的議論の材料

## 現在のプロトタイプ

### `pydanticai_integration/`
PydanticAI統合による WebApiBaseAnnotator の置き換え概念実装。

- **目標**: PydanticAI Agent を使用した新しい WebAPI アノテーター基底クラス
- **関連記録**: Notion LTM (OpenClaw) `pydanticai_implementation_complete_history_2025`
- **実装段階**: ✅ 完了・本番統合済み（2025年10月）

## 使用方針

1. **概念検証**: 新技術・アーキテクチャの実装可能性を検証
2. **プロトタイプ開発**: 本番実装前の試作・実験
3. **段階的統合**: 検証完了後、`src/`ディレクトリに正式移行

## 注意事項

- このディレクトリのコードは**実験的**であり、本番環境での使用は想定していません
- プロトタイプが成熟したら、適切な場所（`src/`等）に移行してください
- 不要になったプロトタイプは定期的にクリーンアップしてください

## プロトタイプの作成ガイドライン

1. 明確なサブディレクトリを作成
2. `__init__.py`でPythonパッケージとして認識
3. 関連するRFC文書への参照を含める
4. 概念実装・テストスクリプトを開発
5. 検証完了後、本番実装に移行
